# Data Science Bootcamp

Hands-on projects and exercises completed during TripleTen Data Science bootcamp, organized by sprints.

## Skills & Tools
- Python (core language fundamentals)
- pandas, NumPy
- Data wrangling & cleaning
- Jupyter Notebooks
- Git & GitHub

## Repository Structure
- `sprints/` – Bootcamp work organized by sprints
  - `sprint_01_python_basico` – Python fundamentals
  - `sprint_02_python_basico_continuacion` – Python fundamentals continuation
  - `sprint_03_manipulacion_de_datos` – Data cleaning and transformation with pandas
  - `sprint_04_manipulacion_de_datos_continuacion` – Data cleaning and transformation with pandas continuation
  - `sprint_05_analisis_estadistico_de_datos` – Exploratory data analysis and statistical concepts
  - `sprint_06_proyecto_modulo_1` – End-to-end data analysis project (Module 1)
  - `sprint_07_herramientas_de_desarrollo_de_software` – Git, GitHub, environments, and software development practices
  - `sprint_08_recopilacion_y_almacenamiento_de_datos` – Data collection from files, APIs, and databases
  - `sprint_09_curso_de_preparacion_de_carrera` –  Career preparation: CV, interviews, and portfolio
  - `sprint_10_introduccion_al_machine_learning` – Machine Learning fundamentals
  - `sprint_11_aprendizaje_supervisado` – Supervised learning models and evaluation
  - `sprint_12_aprendizaje_automatico_en_negocios` – Applying ML to business problems and decision-making
  - `sprint_13_proyecto_del_modulo_2` – Applied Machine Learning project (Module 2)
  - `sprint_14_algebra_lineal` – Linear algebra concepts for Machine Learning
  - `sprint_15_metodos_numericos` – Numerical methods and optimization techniques
  - `sprint_16_series_temporales` – Time series analysis and forecasting
  - `sprint_17_aprendizaje_automatico_de_textos` – Natural Language Processing and text modeling
  - `sprint_18_vision_artificial` – Computer Vision fundamentals and image processing
  - `sprint_19_proyecto_final` – Final capstone project integrating multiple Data Science skills

## How to Use This Repository
Each sprint contains exercises, project, and resources.  
Full projects include their own README with problem description and results.
