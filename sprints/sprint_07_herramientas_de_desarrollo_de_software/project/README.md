# $\textcolor{blue}{Sprint\ 7\ -\ Herramientas\ de\ desarrollo}$
### Descripción del proyecto

Creación y gestión de entornos virtuales de Python y el desarrollo de una aplicación web.

Esta aplicación hecha con Streamlit muestra cómo se pueden construir una web app con Python y ponerla en línea sin necesidad de usar HTML, CSS o Javascript. 
Te permite convertir tu Jupyter notebook en una aplicación con botones, casillas de verificación y gráficas.

URL de la aplicación en streamlit: https://sprint-7-project-jcue.onrender.com/

### $\textcolor{yellow}{Cómo\ correr\ el\ programa\ desde\ local:}$
- Instala pandas, plotly y streamlit

- Desde la carpeta del proyecto ejecuta 'streamlit run app.py'

# $\textcolor{blue}{Sprint\ 7\ -\ Software\ Development\ Tools}$
### Project Description

Creation and management of Python virtual environments and the development of a web application.

This Streamlit application demonstrates how to build and deploy a Python-based web app without using HTML, CSS, or JavaScript. 
It lets you transform a Jupyter notebook into an interactive application with buttons, checkboxes, and charts.

Streamlit app URL (in Spanish): https://sprint-7-project-jcue.onrender.com/

### $\textcolor{yellow}{How\ to\ run\ the\ program\ locally:}$
- Install pandas, plotly and streamlit

- Go to the project's folder and execute 'streamlit run app.py'
