# $\textcolor{blue}{Sprint\ 18\ -\ Visión\ artificial}$
### Descripción del proyecto

Construye y entrena una red neuronal convolucional utilizando el conjunto de datos con fotos de personas. Obtén el valor EAM para el conjunto de pruebas no superior a 8.

Tu script debe contener cuatro funciones:

- load_data(path, subset): carga los conjuntos de datos de entrenamiento y prueba.
- create_model(input_shape): define un modelo.
- train_model(model, train_data, test_data, batch_size, epochs): controla el entrenamiento del modelo. Deberías establecer epochs (épocas) a 3. Tomará algo de tiempo entrenar tu modelo, así que sé paciente.

# $\textcolor{blue}{Sprint\ 18\ -\ Computer\ Vision}$
### Project Description

Build and train a convolutional neural network using the dataset with photos of people. Achieve a MAE value on the test set no greater than 8.

Your script must contain four functions:

- load_data(path, subset): loads the training and test datasets.
- create_model(input_shape): defines the model.
- train_model(model, train_data, test_data, batch_size, epochs): controls the model training process. You should set epochs to 3. Training your model will take some time, so be patient.
